minutes2github

This tool parses HTML minutes and links back from these github issues to the minutes

The form below requires a Github Personal Access Token, which you can obtain from your Github profile settings under "Developer Settings" > "Personal access tokens" > "Tokens (classic)"; it needs the "public_repo" scope.

NB: you can manually verify that this page is not sending or recording this token anywhere, but in general, beware that sharing such a token is sharing a lot of your Github privileges.
